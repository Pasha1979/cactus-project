# 🌍 Инструкция по использованию GBIF интеграции

## 📋 Что такое GBIF и как это работает

**GBIF (Global Biodiversity Information Facility)** - это глобальная база данных о биоразнообразии, которая содержит:
- 📍 **Точки наблюдения** растений в природе с координатами
- 📸 **Фото из природы** от натуралистов и исследователей  
- 🌍 **Данные о странах** произрастания
- 🏞️ **Информацию об ареалах** обитания

## 🔄 Как происходит интеграция

### 1. **Порядок получения данных:**
```
1. Запрос к Llifle API → Основная информация о растении
2. Запрос к GBIF API → Дополнительные данные из природы
3. Объединение данных → Полная информация о растении
```

### 2. **Приоритеты данных:**
- **Страна**: GBIF (более точные occurrence данные)
- **Ареал**: Объединение Llifle + GBIF  
- **Синонимы**: Объединение уникальных
- **Фото**: Llifle (культурные) + GBIF (из природы)

## 🎯 Где найти GBIF данные в приложении

### 📱 **Вкладка "Распространение"**
- 🗺️ **Интерактивная карта** с точками наблюдения
- 📊 **Статистика** occurrence и фото
- 🇺🇸 **Страна** с флагом (из GBIF)
- 🏞️ **Ареал обитания** (объединенный)
- 📸 **Фото из природы** (горизонтальная прокрутка)

### 🖼️ **Вкладка "Галерея"**
- Новый фильтр **"GBIF"** для фото из природы
- Фильтры: **"Все"**, **"Мои"**, **"С Llifle"**, **"GBIF"**

## 🔧 Как добавить/обновить GBIF данные

### 📥 **Автоматическое получение:**
GBIF данные запрашиваются автоматически при:
- ✅ Первом открытии карточки растения
- ✅ Обновлении данных через кнопку 🔄 во вкладке "Распространение"
- ✅ Очистке кэша (7 дней TTL)

### 🔄 **Принудительное обновление:**
1. Откройте карточку растения
2. Перейдите на вкладку **"Распространение"**
3. Нажмите кнопку **🔄** (Обновить данные GBIF)
4. Дождитесь завершения загрузки

## 📊 Что отображается на карте

### 📍 **Точки наблюдения:**
- 🔴 **Красные маркеры** - отдельные точки наблюдения
- 🟢 **Зеленый полигон** - предполагаемый ареал (выпуклая оболочка)
- 📱 **Интерактивность** - клик для детальной информации

### 📋 **Информация о точке:**
- 🇺🇸 Страна наблюдения
- 🏙️ Местность (locality)  
- 🌍 Ареал (habitat)
- 📍 Координаты (lat, lng)
- 📅 Год наблюдения

## 🖼️ Работа с фото GBIF

### 📸 **Где найти фото:**
1. **Вкладка "Распространение"** - горизонтальная галерея
2. **Вкладка "Галерея"** - фильтр "GBIF"

### 🎯 **Особенности фото:**
- 🌿 **Только из природы** (не культурные растения)
- 📱 **Высокого качества** от iNaturalist и других источников
- 🔄 **Автоматическое обновление** при обновлении GBIF данных

## 🛠️ Технические детали

### 📡 **API запросы:**
```
GET https://api.gbif.org/v1/occurrence/search
?scientificName=Astrophytum+asterias
&limit=50
&hasCoordinate=true
```

### 💾 **Кэширование:**
- **TTL**: 7 дней
- **Ключ**: `gbif_data_<latinName>`
- **Хранилище**: SharedPreferences

### 🔍 **Поиск растений:**
Используйте **латинские названия** для лучших результатов:
- ✅ `Astrophytum asterias`
- ✅ `Mammillaria gracilis`  
- ❌ "Астрофитум" (найдет меньше результатов)

## 🚨 Поиск и устранение проблем

### ❌ **Не загружаются GBIF данные:**
1. **Проверьте интернет-соединение**
2. **Используйте латинское название**
3. **Нажмите кнопку обновления 🔄**
4. **Проверьте логи** в консоли Flutter

### ❌ **Не показываются фото GBIF:**
1. **Перейдите на вкладку "Галерея"**
2. **Выберите фильтр "GBIF"**
3. **Обновите данные** через кнопку 🔄
4. **Проверьте наличие** occurrence с фото

### ❌ **Карта не работает:**
1. **Проверьте интернет** (нужны тайлы OpenStreetMap)
2. **Убедитесь** что есть occurrence с координатами
3. **Попробуйте** центрировать карту через кнопку 🎯
